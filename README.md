# KBFlow

Use your MacBook as a keyboard + mouse for a Windows PC connected to your TV.
Intercepts inputs natively on macOS and streams them directly to Windows via TCP for minimal latency. 

## Requirements
- MacBook (macOS 13+)
- Windows 10/11 PC
- Both on the same local Wi-Fi network

## Quick start
1. Download `KBFlowServer.exe` → run on Windows.
2. Open `KBFlow.app` on Mac → enter Windows IP.

To exit full screen mode, **swipe up with 4 fingers** on your MacBook trackpad.

## Keyboard Remapping
By default:
- `Command` maps to Windows `Win` key
- `Option` maps to Windows `Alt`
- `Control` maps to `Ctrl`

See `docs/setup-guide.md` for full installation instructions, including firewall setup.
